# 🚀 Tech Career Navigator

Tech Career Navigator is a Streamlit-based web application that helps students and early-career learners discover the technology domain best suited to their interests and strengths through a personalized assessment.

Instead of choosing a career path based on trends or assumptions, users complete a short quiz and receive personalized recommendations, compatibility scores, and a structured learning roadmap to help them begin their journey.

---

## ✨ Features (Version 1.0)

* 🎯 Domain recommendation quiz
* 📊 Percentage-based compatibility scores
* 🗺️ Personalized learning roadmap
* 💻 Programming language recommendations
* 🛠️ Essential tool recommendations
* 🎓 Beginner-friendly free course suggestions
* 🚀 Project ideas for hands-on learning
* 🔄 Retake quiz functionality

## ▶️ Running the Project

1. Clone the repository.
2. Install the required package:

```bash
pip install -r requirements.txt
```

3. Run the application:

```bash
streamlit run app.py
```

## 💡 Motivation

Choosing the right technology career path can be overwhelming for students due to the wide range of available domains. Tech Career Navigator aims to simplify this decision by providing structured, personalized guidance and practical learning resources.
